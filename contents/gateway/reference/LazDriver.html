LazDriver インストール方法

LazDriverはLazurite Pi Gateway用のカーネルドライバです。

#以降はコメントなので入力する必要はありません

1) Lazurite Pi Gateway用ドライバファイルをダウンロードするディレクトリに移動します。

以下の赤字部分を実行していきます。

pi@raspberrypi ~ $ cd driver/      #driverのディレクトに移動します

2) サーバーからドライバを取得します。

pi@raspberrypi ~/driver $ git clone git://github.com/LAPIS-Lazurite/LazDriver.git    #GitHubのサーバーから無線モジュール用ドライバファイルを取得します
Cloning into 'LazDriver'...
remote: Counting objects: 55, done.
remote: Compressing objects: 100% (35/35), done.
remote: Total 55 (delta 16), reused 55 (delta 16), pack-reused 0
Receiving objects: 100% (55/55), 128.18 KiB | 203 KiB/s, done.
Resolving deltas: 100% (16/16), done.

3) ダウンロードしたフォルダに移動し、カーネルドライバをコンパイルします。

pi@raspberrypi ~/driver $ cd LazDriver/    #ダウンロードしたドライバファイルのフォルダに移動します
pi@raspberrypi ~/driver/LazDriver $ make    #ドライバファイルをビルドします。
