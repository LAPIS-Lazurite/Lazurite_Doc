 getMyAddr64()


詳細

  クラス
namespace		 関数名
共有ライブラリ lazurite int lazurite_getMyAddr64(uint8_t *address)
Ruby LazGem LazGem.getMyAddr64()
Python PyLaz 非対応
JAVA Liblzaurite 非対応

 

機能

無線モジュールが保持している64bit MACアドレスをBig Endianで読み出します

 

パラメータ

*address  (共有ライブラリ)

アドレスを格納する8バイトの配列の先頭ポインタ(unsigned char型)を指定します。

戻り値

共有ライブラリ、JAVA

0        成功
0以外  失敗

LazGem

64bitのアドレスが返ります

例外(Javaのみ)

IOException　　失敗
