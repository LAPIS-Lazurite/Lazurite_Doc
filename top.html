Lazuriteについて

LazuriteはIoTを簡単に実現することができる開発キット／リファレンスデザインです。

回路はオープンハードウエア、ソフトウエアもオープンソースで開発しています。
Lazurite IDE、Rasberry Pi用の無線モジュールドライバソフトの双方ともGPLまたはLGPLのライセンスに基づき配布しています。

 

 

 

Lazurite IDE

1. ソフトウエアのダウンロード方法

Lazuriteのダウンロードサイトよりインストーラーをダウンロードしてください。

インストーラーのファイル名は、setup_001001_2015Jun15.exe です。

2 Lazurite IDEでダウンロードしたソフトウエアのライセンス情報

Lazurite Sub-GHz用のファームウエアである、LazuriteIDE\hardware、LazuriteIDE\librariesは、マイコン用のファームウエアです。マイコン用のファームウエハは、LGPL(Lesser GPL)に基づき配布しています。
LazuriteIDE\BINのフォルダ内のファイルは、それぞれのソフトウエアのライセンスに基づき再配布しています。

Lazurite Pi Gateway

ご使用する前に

Lazurite Pi Gatwayを使用するためには、次のものを揃える必要があります。

  1. Raspberry Pi (B+, A+)またはRaspberry Pi2　(26pinのRaspberry Pi Bには対応していません
  2. Raspberry Pi用の備品 (電源、マイクロUSBコネクタ、８GB以上のマイクロSDカード、モニター、キーボード、マウス等）

これは弊社の開発環境です。

 

ソフトウエアのダウンロード

1. Lazuriteのダウンロードサイトより、Lazurite Pi Gateway用のイメージファイルをダウンロードしてください。

◆Raspberry Pi別に２種類のイメージファイルがあります。

Raspbery Pi A+ / B+用 150318_RPi_LAZURITE.zip
Raspberyy Pi2用 150401_RPi2_LAZURITE.zip

2. Raspberry Pi用ソフトウエアのライセンスについて
