GPIO

概要

GPIOはGereneral Purpose Input Output(汎用入出力端子)の略で、ほとんどのマイコンの端子がこの機能を搭載しています。

以下の関数によって、pinMode関数によって端子を入力モードにすると、digitalRead関数で端子の状態を読み出すことができます。また、出力モードのときは端子からHIGH/LOWレベルを出力することができます。

このモードで読み取ったり、出力したりできるのは、HIGHまたはLOWのみです。

アナログ信号を入出力するときは、analogReadやanalogWriteの関数を使用してください。

 

関数

void pinMode(pin, mode)

void digitalWrite(pin, val)

int digitalRead(pin)
