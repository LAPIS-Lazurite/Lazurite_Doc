LazuriteJavaのインストール方法

LazuriteJavaはLazurite Pi Gateway用のJava Libraryです。

 LazuriteJavaのインストール

#の後ろはコメントなので実行する必要はありません。
赤字部分を実行してください。
ここでは、ホームディレクトリの下の"java"フォルダで作業しています。

1) ソースコードを取得します。

git clone git://github.com/LAPIS-Lazurite/LazuriteJava⏎

2) JAVAのライブラリをインストールします。

cd LazuriteJava⏎           #フォルダの中に移動します

pi@raspberrypi ~/java/LazuriteJava $ ls⏎        #フォルダの中身を確認すると2つのファイルと、2つのフォルダがあります。
Makefile README.md lib sample

pi@raspberrypi ~/java/LazuriteJava $ make    #プログラムをビルドします。
/usr/bin/javac -g -Xlint lib/com/lapis_semi/lazurite/io/SubGHzEventListener.java lib/com/lapis_semi/lazurite/io/sample_serial.java lib/com/lapis_semi/lazurite/io/sample_raw2.java lib/com/lapis_semi/lazurite/io/common.java lib/com/lapis_semi/lazurite/io/mac_info.java lib/com/lapis_semi/lazurite/io/RawEventObject.java lib/com/lapis_semi/lazurite/io/SubGHz.java lib/com/lapis_semi/lazurite/io/sample_raw.java lib/com/lapis_semi/lazurite/io/RawEventListener.java lib/com/lapis_semi/lazurite/io/SubGHzEventObject.java lib/com/lapis_semi/lazurite/io/Raw.java lib/com/lapis_semi/lazurite/io/CtrlDeviceList.java

pi@raspberrypi ~/java/LazuriteJava $ make install        #ビルドした結果からjarファイルを作成し、javaのフォルダにインストールします。
jar cvf SubGHz.jar -C lib/ com

サンプルプログラムの実行

1) サンプルプログラムが格納されているフォルダに移動します。

pi@raspberrypi ~/java/LazuriteJava $ cd sample/⏎
pi@raspberrypi ~/java/LazuriteJava/sample $ ls                            #ビルド用のmakefileと3種類のサンプルプログラムがあります。
Makefile sample_raw.java sample_raw2.java sample_serial.java

pi@raspberrypi ~/java/LazuriteJava/sample $ make⏎                  #サンプルプログラムをビルドします。
/usr/bin/javac -g -Xlint ./sample_serial.java ./sample_raw2.java ./sample_raw.java

2) サンプルプログラムの実行方法

送信用サンプルプログラム        作成未
受信用サンプルプログラム
・sample_raw.rb
　パケット受信のサンプルプログラムの例です。  mainルーチンで常時受信用バッファを監視しています。
・sample_raw2.rb
　パケット受信のサンプルプログラムの例です。Rawレベルドライバから受信時にイベントが発生する例です。
・sample_serial.rb
　固定の送信アドレスからのデータだけを受信するモードのサンプルです。
   シリアルライブラリに似たインタフェースのため扱いやすいモードです。
