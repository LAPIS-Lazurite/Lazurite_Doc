ble.print_long (long data, unsigned char fmt)
ble.println_long(long data, unsigned char fmt)


詳細

void ble.print_long(long data, unsigned char fmt);
void ble.println_long(long data, unsigned char fmt);

仕様

ble.print_long関数は、long型データを数値の文字列に変換してBluetoothSMARTで送信する関数です。
ble.println_long関数は、ble.print_long関数で作成した文字列の最後に改行コードを付与します。

本関数を実行したら送信するデータをバッファに格納し、送信を開始します。
全データがバッファに格納できたら送信が終わる前に本関数は終了し、バックグラウンドで1バイトずつデータの送信を行います。
バッファがフルになった時は、全データがバッファに格納できるまで待って本関数が終了します。

パラメータ

    long data

文字列に変換するデータです。

    unsigned char fmt

変換するデータの書式を示します。

  BIN 2進数で表示
OCT 8進数で表示
  DEC 10進数で表示
  HEX 16進数で表示

戻り値

なし
