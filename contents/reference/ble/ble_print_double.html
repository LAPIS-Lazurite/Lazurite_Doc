int BLE.print_double (data, digit)
int BLE.println_double(data, digit)


詳細

int BLE.print_double(double data, unsigned char digit);
int BLE.println_double(double data, unsigned char digit);

仕様

BLE.print_double関数は、浮動小数点 double型データを数値の文字列に変換してBluetoothSMARTで送信する関数です。
BLE.println_double関数は、Serial.println_double関数で作成した文字列の最後に改行コードを付与します。

本関数を実行したら送信するデータをバッファに格納し、送信を開始します。
全データがバッファに格納できたら送信が終わる前に本関数は終了し、バックグラウンドで1バイトずつデータの送信を行います。
バッファがフルになった時は、全データがバッファに格納できるまで待って本関数が終了します。

パラメータ

   double data

文字列に変換する浮動小数点 double型データです。

    unsigned char digit

 表示する小数点の桁数を指定します。

戻り値

0> 送信した文字数を返します。

0< エラー状態を返します。
　-1 ~ -4：　シリアル送信不可能なモード（mode * -1)が返ります。
　-100～：　パラメータのエラー
　-200～：　無線モジュールとの通信エラーが発生したため、モジュールをリセットして復帰した
