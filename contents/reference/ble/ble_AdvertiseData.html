BLE.setAdvertiseData(*data, len)
BLE.getAdvertiseData(*data,*len)


詳細

int BLE.setAdvertiseData(*data, len);
int BLE.getAdvertiseData(*data,*len);

仕様

BLE.setAdvertiseData関数はアドバタイズのデータを設定します。
BLE.getAdvertiseData関数は現在のアドバタイズのデータを取得します。

データは最大で32バイトのバイナリが設定可能で、設定するデータ長をlenで指定します。

パラメータ

   char *data

設定値はアドバタイズのデータが格納されたポインタを指定します。
取得時はアドバタイズのデータを格納するメモリのポインタを指定します。

   char len (設定時のみ)、char *len (取得時のみ)

設定時はアドバタイズデータの長さを指定します。
取得時はアドバタイズデータの長さを格納するメモリのポインタを指定します。

戻り値

0= 設定成功

0< エラー状態を返します。
　-2 ~ -4：　シリアル送信不可能なモード、（mode * -1)が返ります。
　-100～：　パラメータのエラー
　-200～：　無線モジュールとの通信エラーが発生したため、モジュールをリセットして復帰した
