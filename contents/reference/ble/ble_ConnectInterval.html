BLE.setConnectInterval(fast,slow,time)
BLE.getConnectInterval(*fast,*slow,*time)


詳細

int BLE.setConnectInterval(fast,slow,time);
int BLE.getConnectInterval(*fast,*slow,*time);

仕様

BLE.setConnectInterval関数はアドバタイズの間隔を設定します。
BLE.getConnectInterval関数は現在のアドバタイズの間隔を取得します。

状態が変化した時、まずはfastにより設定された時間間隔で接続を開始し、timeの時間が経過するまで通信が無ければslowで設定された通信間隔に移行します。

パラメータ 設定範囲 初期値
fast 8-4000 40 高速動作時のコネクション・インターバルです。
slow 10-4000 100 低速動作時のコネクション・インターバルです。
time 0,500-18000 3000 高速動作から低速動作に移行する時間を指定します。
0にすると切り替え機能が無効になります。

パラメータ

   int fast (設定時のみ)、int *fast (取得時のみ)

設定時は高速動作時のコネクション・インターバルを指定します。取得時は高速動作時のアドバタイズ間隔を格納するポインタを指定します。

   int slow (設定時のみ)、int *slow (取得時のみ)

設定時は低速動作時のコネクション・インターバル間隔を指定します。取得時は低速動作時のコネクション・インターバル間隔を格納するポインタを指定します。

   int time (設定時のみ)、int *time (取得時のみ)

設定時は高速動作から低速動作に移行する時間を指定します。取得時は取得した値を格納するポインタを指定します。

戻り値

0= 設定成功

0< エラー状態を返します。
　-2 ~ -4：　シリアル送信不可能なモード、（mode * -1)が返ります。
　-100～：　パラメータのエラー
　-200～：　無線モジュールとの通信エラーが発生したため、モジュールをリセットして復帰した
